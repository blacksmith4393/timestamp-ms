Zach's Timestamp Microservice

Pass a string, containing either a unix timestamp or a natural language date,
and the server will return both the Unix timestamp and the natural language form of that date.

Example Usage:

http://localhost:5000/1450137600
http://localhost:5000/December%2015,%202015

Example Output:

{ "unix": 1450137600, "natural": "Tue Dec 15 2015" }
